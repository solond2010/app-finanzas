# Logos de banco

Deja aquí los logos de los bancos/fintech para las cuentas.

Formato:
- Preferible **SVG** (o PNG con fondo transparente, ~128x128).
- Nombre del archivo = slug en minúsculas, sin espacios ni acentos.

Ejemplos:
- revolut.svg
- bbva.svg
- santander.svg
- caixabank.svg
- n26.svg
- traderepublic.svg
- myinvestor.svg
- ing.svg

Una vez estén aquí, se añade el selector de logo en el diálogo de cuenta
y se renderizan en el dashboard y en /cuentas.
